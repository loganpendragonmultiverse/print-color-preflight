# Support

Use GitHub Discussions for workflow questions and GitHub Issues for reproducible defects or focused feature proposals.

When asking about output, include the application version, operating system, Ghostscript version, rendering intent, destination profile description and hash, input preflight JSON, and sanitized reproduction steps. Do not upload customer artwork or a profile you lack permission to redistribute.

The project cannot select a printer profile, certify a press, calibrate a display, approve a physical proof, or guarantee acceptance by a particular printer. Those decisions belong to the production provider.
